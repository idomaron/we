How to Install

To install the plugin all you need to do is use the built-in wordpress plugin upload tool. Upload the plugin zip and activate the plugin, from there you can go to the settings page and get started.

Settings Page

General Settings

Use Short Links in Emails - Enable this feature to generate shortened goo.gl URLs to use in your emails for verification.

Goo.gl API Key - Enter an optional goo.gl API key for tracking.

Fake Email Pentaly - Enter the number of minuets users should wait if they entered a fake email they could not verify. Default is 3.

 

Edit Templates

From here you can edit the templates that appear in the various stages of the plugin. Each is pretty simple to figure out, the preloaded text should get the job done!

How to Use

From your post / page editor you will see a new email icon in your tinyMCE toolbar. This MCE plugin will generate the shortcode for you. All you need to do is insert the download URL and the rest will be taken care of for you.

Keep in mind that download URLs should push the file to the user, or you could use an uploading service.

If the URL does not start a download, the user will instead be redirected to the URL you provide.
